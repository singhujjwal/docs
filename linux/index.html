Setting proxy for updates

Ubuntu

cat /etc/apt/apt.conf

Acquire::http::Proxy  "http://proxy.example.com:80";
Acquire::ftp::Proxy   "https://proxy.example.com:80";
Acquire::https::Proxy "http://proxy.example.com:80";
